웹 접근성

웹 접근성

웹에 액세스(접근)하는 사람과 방법에 관계없이 모든 사용자가 신체적, 환경적 조건에 관계없이 웹을 동일하게 접근할 수 있도록 제작하는 방법. 단순히 장애인에 대한 것이라고 생각하지만, 정상적인 웹 사용이 가능한 일반 사용자부터, 고령자, 신체적 장애로 특수한 환경에서 접속해야 하는 사용자 등이 어떠한 브라우저를 선택해 사용하던 어떠한 기기로 웹을 이용하던 어떠한 도구로 접근을 하던 네트워크 연결이 느린 환경이던 빠른 환경이던 정보제공이 동일하게 이루어져야함. 이런 사용자와 환경들을 고려해서 대체 텍스트를 제공하고 논리적인 아웃라인을 만들며 탭키로 모든 링크와 이미지 등에 논리적인 순서대로 접근이 가능하게 만드는 것과 크로스 브라우징은 기본으로 깔고 웹을 만들어야함. 접근성이 높은 웹을 만들기 위해서는 자신의 컴퓨터와 웹 사용 방식을 넘어서 생각하고 다른 사람이 웹을 사용하는 방식에 대해서 많은 생각을 해야함. 접근성을 향상시키는 시맨틱한 마크업은 SEO(검색 최적화)를 향상시켜 사이트를 더 쉽게 찾을 수 있도록 만들어주는 등 접근성을 고려해 만든 사이트는 많은 이용자를 확보할 수 있고 속도와 성능도 빠름.
접근성 문제가있는 기존 웹 사이트를 개조하는데에는 많은 비용이 들기 때문에 프로젝트 시작부터 고려해야함.
Firefox 접근성 검사기는 접근성 문제를 확인하는 데 매우 유용한 도구.
